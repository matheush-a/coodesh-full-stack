# Onfly (quasar-project)

A Quasar Project

## Instale as dependências
```bash
yarn
# ou
npm install
```

### Execute em modo de desenvolvimento
```bash
yarn dev
# ou
npm run dev
```


### Verifique a qualidade do código
```bash
yarn lint
# ou
npm run lint
```


### Formate os arquivos
```bash
yarn format
# ou
npm run format
```



### Gere o aplicativo para modo de produção
```bash
quasar build
```
