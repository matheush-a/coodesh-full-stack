# Onfly (quasar-project)

A Quasar Project

## Instale as dependências
```bash
yarn
# or
npm install
```

### Execute em modo de desenvolvimento
```bash
quasar dev
```


### Verifique a qualidade do código
```bash
yarn lint
# or
npm run lint
```


### Formate os arquivos
```bash
yarn format
# or
npm run format
```



### Gere o aplicativo para modo de produção
```bash
quasar build
```
